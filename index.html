Snapshot history

GitHub's API only exposes a cumulative download count — no history. Release Radar fixes that with zero backend: every time you analyze a repo, a timestamped snapshot is saved locally in your browser. Come back tomorrow, next week, next month — and the tool computes real measured velocity and plots true downloads-over-time. Export the history as JSON to back it up, sync across machines, or share it.

Release Radar — a static, backend-free GitHub release analytics tool.
All processing happens in your browser. Snapshots are stored locally and never leave your machine unless you export them.
